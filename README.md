# Godot-tdlib
GDExtension C++ library for working with TdLib (JSON interface) on Godot Engine

### Realized Functions
1. `TdJson.send`
2. `TdJson.receive`
3. `TdJson.destroy`
4. `TdJson.execute`

### Functions in process
1. `TdJson.set_log_callback`
2. Something else...

## Usage
This library can be used to create games or applications for integration or create alternative Telegram clients through TelegramApi
